## Project Description

This project implements a classic **Memory Game** using **SwiftUI**. The game features a grid of cards that the user flips to find matching pairs. This version includes a `ProgressView` to visually track the player's progress as they match cards.

## How to Open the Project

1.  Open **Xcode**.
2.  Go to **File > Open...**.
3.  Navigate to the `Homework13_Stephen_David` folder and select the `Homework13_Stephen_David.xcodeproj` file.

## How to Run the Project

1.  Once the project is open in Xcode, select a simulator (e.g., **iPhone 15 Pro**) from the scheme dropdown menu at the top of the Xcode window.
2.  Click the **Run** button (the play icon) in the toolbar, or press `Command + R`.
3.  The app will build and launch in the selected simulator.

## Assignment Details

This assignment required integrating a `ProgressView` into the Memory Game. The progress bar updates dynamically, showing the percentage of cards that have been successfully matched.
